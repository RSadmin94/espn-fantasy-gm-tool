# Verified manual league history

Place `ATLANTAS_FINEST_FF_History.xls` (or `.xlsx`) in this folder **or** set:

`VERIFIED_MANUAL_HISTORY_XLS` to the absolute path of your workbook.

The server reads these sheets (case-insensitive names):

- Champions  
- Current Teams Records  
- All Teams Records  
- H2H Matrix  
- H2H Detailed Matchups  

Optional: any sheet whose header row looks like a draft recap (Season, Round, Player, …).

Season **2009** is ignored for manual reads.
